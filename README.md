# PAR-C® Observación

Aplicación web estática preparada para publicar en GitHub Pages.

## Publicación rápida

1. Crea un repositorio en GitHub llamado `parc-observacion`.
2. Sube todos los archivos de esta carpeta, no el ZIP.
3. Comprueba que `index.html` queda en la raíz del repositorio.
4. Entra en `Settings > Pages`.
5. En `Source`, selecciona `Deploy from a branch`.
6. En `Branch`, selecciona `main` y carpeta `/root`.
7. Pulsa `Save`.

La web quedará publicada en una URL similar a:

`https://TU-USUARIO.github.io/parc-observacion/`
